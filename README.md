# OCLC Cataloging Widget

## Instructions on how to use:

1. Download the source code zip file.
2. Extract the zip file using file explorer into your preferred directory.
3. Locate the OCLC Cataloging Widget folder in that directory.
4. Locate the setup folder within the root folder, and run setup.cmd.
5. Follow the setup instructions, for which you will need a WorldCat API Key, API secret, and an API scopes.
6. After that completes, exit and go to the root directory and find the app folder.
7. Run main.cmd, and the widget will pop up!
